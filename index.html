Help Guide for To-Do List




-This is a To-Do list site, where you can insert the tasks you want to do, and then check it when its done;

-To start, click on "+" button to insert a noe box of task, you will need to type the title of the box and press enter;

-Then, you can text the first task and "enter" again, keep doing that until all tasks are all inserted;

-You can also insert another box by hiting "+" again. You can insert how many box and tasks you want to;

-The tasks are editable, you can delete then from the box by leaving it empty and hitting "backspace" or using the "edit_mode";

-When "edit mode" is activated, "X's" will display aside every task and title, by clicking them you will dele the respective task/title;

-The X on the bottom will delete the box;

-If there isn't any empty task spot to type on, you must enter on the last task and "enter" while in it, a new empty task must appear;

-When a task is marked as done, by checking the checkbox, the task text get crossed through;

-When every task gets checked, the box gets green indicating all the tasks have been done;

-You can also type some word (or part of a word) in the search bar, and only boxes with that content will display on your screen;

-To all boxes appear again, just clear the search bar.

-To close this guide click on the "X" button or just click somewhere outside the guide.
